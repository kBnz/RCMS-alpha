get_config_vars()

get_config_vars() -- 読み込まれた設定ファイル変数を返します。

説明

array get_config_vars ( [string varname])

パラメータが与えられない場合は 全ての読み込まれた 設定ファイル変数 の配列が返されます。

例 13-1. get_config_vars()

<?php

// 読み込まれた設定ファイル変数'foo'を取得します
$myVar = $smarty->get_config_vars('foo');

// 全ての設定ファイル変数を取得します
$all_config_vars = $smarty->get_config_vars();

// では見てみましょう
print_r($all_config_vars);
?>

clear_config()、 {config_load}、 config_load() および get_template_vars() も参照してください。
