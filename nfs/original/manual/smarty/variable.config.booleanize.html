$config_booleanize

TRUE の場合、設定ファイル の on/true/yes や off/false/no といった値が自動的に boolean 値に変換されます。これにより、テンプレートで {if #foobar#}...{/if} のように使用できるようになります。foobar が on、true あるいは yes である場合に {if} ステートメントを実行します。 デフォルトは TRUE です。
