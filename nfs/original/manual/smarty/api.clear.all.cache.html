clear_all_cache()

clear_all_cache() -- 全てのテンプレートのキャッシュをクリアします。

説明

void clear_all_cache ( [int expire_time])

任意のパラメータとして、キャッシュファイルを削除する前にそのファイルが存在しなくてはならない 最低限の時間（秒）を与える事が出来ます。

例 13-1. clear_all_cache

<?php
// キャッシュ全体をクリアします
$smarty->clear_all_cache();

// 一時間以上経過しているファイルをすべてクリアします
$smarty->clear_all_cache(3600);
?>

clear_cache()、 is_cached() および キャッシュ のページも参照してください。
