$error_reporting

この値に null でない値がセットされると、その値は display() と fetch() の内側で PHP の error_reporting レベルとして使用されます。デバッグ が有効のときはこの値は無視され、エラーレベルには全く触れられません。

trigger_error()、 デバッギングコンソール および トラブルシューティング も参照してください。
