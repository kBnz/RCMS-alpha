register_prefilter()

register_prefilter() -- プリフィルタを動的に登録します。

説明

void register_prefilter ( mixed function)

テンプレートをコンパイルする前に実行する、 プリフィルタ を動的に登録します。プリフィルタ関数の定義の仕方は、プリフィルタ の項を参照して下さい。

PHP 関数のコールバック function は、次のいずれかとなります。

  • 関数名を含んだ文字列

  • array(&$object, $method) 形式の配列 （&$object はオブジェクトの参照で、 $method はメソッド名を含む文字列）

  • array($class, $method) という形式の配列 （$class はクラス名であり、 $method はクラスのメソッド）

unregister_prefilter()、 register_postfilter()、 register_ouputfilter()、 load_filter()、 $autoload_filters および アウトプットフィルタ も参照してください。
